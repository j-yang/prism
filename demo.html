🌟 PRISM 服务器文件浏览器

类似 WinSCP 的网页文件管理工具

在您的网页应用中实现服务器文件浏览、上传和管理功能

🔐 服务器连接

  • 支持 SFTP 和 FTP 协议
  • 安全的用户认证
  • 连接状态管理
  • 自动重连机制

📁 文件浏览

  • 类似 Windows 资源管理器的界面
  • 文件和文件夹图标识别
  • 面包屑导航
  • 文件大小和修改时间显示

📤 文件上传

  • 拖拽上传支持
  • 多文件同时上传
  • 上传进度显示
  • 错误处理和重试

🛠️ 文件管理

  • 创建新文件夹
  • 删除文件和文件夹
  • 文件下载
  • 文件操作历史

📋 使用说明

1 启动应用
npm run dev

在浏览器中打开 http://localhost:5173

2 切换到文件浏览器

点击顶部导航栏的 "File Browser" 标签

3 连接服务器

填写服务器连接信息：

  • 协议：选择 SFTP 或 FTP
  • 主机：服务器IP地址或域名
  • 端口：默认22（SFTP）或21（FTP）
  • 用户名：您的服务器用户名
  • 密码：您的服务器密码

点击"连接"按钮建立连接

4 浏览文件

连接成功后，您可以：

  • 双击文件夹进入子目录
  • 使用面包屑导航快速跳转
  • 查看文件详细信息
  • 选择多个文件进行操作
5 上传程序文件

将生成的 SAS 程序上传到服务器：

  • 拖拽文件到上传区域
  • 或点击"选择文件"按钮
  • 查看上传进度
  • 确认上传成功

⚙️ 技术实现

1 服务器文件管理器

核心服务类 ServerFileManager 处理所有文件操作：

  • 连接管理
  • 文件列表获取
  • 文件上传下载
  • 目录操作
2 Vue 组件

ServerFileBrowser.vue 提供用户界面：

  • 响应式设计
  • 实时状态更新
  • 拖拽支持
  • 错误处理
3 模拟 API 服务

MockServerAPI.ts 模拟后端功能：

  • 文件系统模拟
  • API 请求拦截
  • 异步操作处理
  • 错误模拟

🚀 生产环境部署

重要提醒： 当前实现使用模拟服务器进行演示。在生产环境中，您需要实现真实的后端服务。
1 后端服务器

创建支持以下功能的后端 API：

  • SFTP/FTP 连接管理
  • 文件系统操作
  • 用户认证和授权
  • 文件上传下载处理
2 安全配置

确保以下安全措施：

  • HTTPS 加密传输
  • CORS 策略配置
  • 文件类型限制
  • 访问权限控制
3 性能优化

考虑以下优化策略：

  • 文件分片上传
  • 断点续传
  • 连接池管理
  • 缓存策略
